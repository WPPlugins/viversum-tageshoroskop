viversum Tageshoroskope für alle Sternzeichen als Widget

== Description ==
Stellt das [Tageshoroskop von viversum](http://www.viversum.de/widgets/tageshoroskop) für alle zwölf Sternzeichen als Wordpress-Widget zur Verfügung.

== Installation ==
= Automatisierte Installation =
Im Wordpress-Adminbereich unter "Plugins -> Installieren" nach "viversum Tageshoroskop" suchen und installieren.

= Manuelle Installation =
"viversum-tageshoroskop.zip" entpacken und den Ordner "viversum-tageshoroskop" in das Verzeichnis "/wp-content/plugins/" hochladen oder das ZIP-Paket via "Plugins > Installieren > Hochladen" im Wordpress-Adminbereich hochladen

= Aktivieren des Widgets =
Aktivieren Sie das Plugin unter "Plugins" im Wordpress-Adminbereich

= Widget zur Seitenleiste hinzufügen =
Nach Aktivierung des Plugins erscheint das Widget im Wordpress-Adminbereich unter "Design -> Widgets". Ziehen Sie das Widget wie gewohnt an die Stelle in Ihrer Seiteleiste, an der das Tageshoroskop erscheinen soll.
